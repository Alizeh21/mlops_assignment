# MLOps Kubeflow Pipeline Assignment

## **Overview**
This project implements a complete MLOps workflow using:

- Git & GitHub  
- Python  
- DVC (Data Version Control)  
- Kubeflow Pipelines (Minikube)  
- Kubernetes + Kubectl  
- Docker  
- Jenkins / GitHub Actions  
- Scikit-learn  

The main goal is to build an end-to-end ML pipeline with **data versioning**, **componentized pipelines**, **model training**, and **CI/CD automation**.
